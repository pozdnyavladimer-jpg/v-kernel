Manifest

We Reject

We reject the idea that computation must be:

- sequential
- instruction-driven
- centrally controlled
- fixed in structure

This model has reached its limits.

---

We Believe

We believe that computation should:

- adapt
- evolve
- self-organize
- stabilize through interaction

Not everything should be executed.

Some things should emerge.

The System
