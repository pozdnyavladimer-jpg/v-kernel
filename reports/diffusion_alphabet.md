# Diffusion Alphabet Report

## Status

Prototype completed.

## Core idea

Reaction-diffusion patterns can be encoded as a morphological alphabet.

Pipeline:

physical dynamics → geometric features → symbol classes

## Discovered alphabet

A = blob / smooth core  
B = spots / ring fragments  
C = mixed spots / transition  
D = symmetry class  

## D sub-alphabet

D0 = flower / radial petals  
D1 = mandala / ring / holes  
D2 = grid / field lattice  

## Meaning

This demonstrates that diffusion fields can produce stable symbolic families without manually drawing symbols.

The system acts as a prototype physical alphabet encoder.
